Culture Transformation

Organizational Culture Change

The Negating Values Model – My Latest Development

👥

For Whom?

  • Growing organizations
  • Organizations in transition
  • Organizations that haven't touched culture and values in a long time
⏱️

When?

  • When organizational culture is a list of values on a designed slide and not real action principles
  • When there are gaps between declared values and actual values
  • When our values are those of the founders but haven't been updated for a long time
  • When culture is a collection of contradictory values that compete with each other
  • When organizational culture is siloed and each unit has a sub-culture that doesn't connect to the whole
📅

Duration

6 Months

On redesigning the organizational culture

5

Developing Derived Applications
